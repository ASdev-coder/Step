Змінна - зберігає дані, вона повинна починатися з літери або _, ім'я має бути унікальним та описувати суть значення
Ім'я змінної не повинно збігатися із зарезервованими ключовими словами у Python (синій колір)

Типи даних
==========

int зберігає ціле значення (44)
float зберігає речове/неціле значення (12.75)
str зберігає текстове значення ('abc')
bool зберігає логічне значення (True/False)

None відсутність значення
Перетворення типів - це означає з одного типу перехід в інший (str > int)

Арифметичні оператори
=======================
+ додавання
- віднімання
* множення
/ поділ
** зведення в ступінь
// розподіл націло (7//3=2)
% залишок від розподілу (7%2=1)


Скорочені форми запису
========================
x = x * 2 АБО x * = 2
sheep = sheep + 1 АБО sheep + = 1
